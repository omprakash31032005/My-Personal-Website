# 🌐 My Personal Website

A simple and well-structured **HTML5 Personal Portfolio Website** developed using only **HTML**. This project showcases my personal profile, interests, and favorite collections while demonstrating the fundamentals of front-end web development through semantic HTML elements.

## ✨ Features

* 👤 About Me
* 🚗 My Favourite Cars
* 🎮 My Favourite Games
* 🍕 My Favourite Foods
* 🌍 My Dream Places
* 🏍️ My Favourite Bikes
* 📞 Contact Information
* 🧭 Navigation Menu
* 🖼️ Images and Media
* 📋 Lists and Tables
* 🔗 Hyperlinks
* 📝 HTML Forms
* 🏗️ Semantic HTML Structure

## 🛠️ Technologies Used

* HTML5

## 🚀 How to Access the Project

### Option 1: Run Locally

1. Clone the repository.

```bash
git clone <repository-url>
```

2. Open the project folder.

3. Double-click **index.html** or open it in any modern web browser such as Chrome, Edge, Firefox, or Safari.

### Option 2: Download ZIP

1. Click **Code → Download ZIP** on GitHub.
2. Extract the ZIP file.
3. Open **index.html** in your preferred web browser.
